## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```

Build documentation,

```bash
$ yarn run docs:build
```

Run test,

```bash
$ yarn test
```

Build library via `father-build`,

```bash
$ yarn run build
```
